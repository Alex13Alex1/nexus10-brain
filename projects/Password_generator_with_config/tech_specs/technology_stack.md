# Техническая спецификация для Password Generator с настраиваемой длиной и сложностью

## 1. Библиотеки Python с версиями

- **Flask (v2.2.5)**: Для создания RESTful API, который будет принимать параметры длины и сложности и возвращать сгенерированный пароль.
- **Flask-RESTful (v0.3.9)**: Расширение для Flask, упрощающее создание RESTful API.
- **Redis (v4.5.1)**: Для кэширования часто используемых паролей или шаблонов, чтобы уменьшить нагрузку на API.
- **Python-dotenv (v1.0.0)**: Для управления конфигурацией приложения через переменные окружения.
- **PyJWT (v2.6.0)**: Для обеспечения безопасности API через токены аутентификации.
- **passlib (v1.7.4)**: Для генерации и валидации паролей с различными уровнями сложности.
- **random (встроенная библиотека Python)**: Для генерации случайных символов.

## 2. API

- **RESTful API**: 
  - **Endpoint**: `/generate-password`
  - **Метод**: POST
  - **Параметры**: 
    - `length` (int): Длина пароля.
    - `complexity` (str): Уровень сложности (например, "low", "medium", "high").
  - **Ответ**: JSON объект с сгенерированным паролем.

## 3. Форматы данных

- **JSON**: Используется для передачи данных между клиентом и сервером через API. Это стандартный формат для RESTful API, обеспечивающий легкость интеграции и обработки данных.

## 4. Обоснование выбора

- **Flask и Flask-RESTful**: Эти библиотеки обеспечивают легковесное и простое в использовании решение для создания RESTful API, что идеально подходит для небольших проектов, таких как генератор паролей.
- **Redis**: Использование Redis для кэширования позволяет значительно снизить нагрузку на API, ускоряя доступ к часто используемым паролям и шаблонам.
- **Python-dotenv**: Упрощает управление конфигурацией приложения, особенно в средах разработки и производства.
- **PyJWT**: Обеспечивает безопасность API, защищая его от несанкционированного доступа.
- **passlib**: Предоставляет мощные инструменты для генерации и валидации паролей, поддерживая различные алгоритмы и уровни сложности.
- **random**: Встроенная библиотека Python, которая позволяет генерировать случайные символы, необходимая для базовой генерации паролей.

Эти технологии и библиотеки были выбраны с учетом их стабильности, производительности и способности удовлетворять требованиям системы, обеспечивая при этом безопасность и оптимизацию ресурсов.
