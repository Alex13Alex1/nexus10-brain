### Техническая спецификация для скрипта мониторинга цены BTC

#### Требуемые библиотеки и инструменты:

1. **Python 3.x**:
   - Необходим для выполнения скрипта.

2. **Библиотеки Python**:
   - `requests`: Для выполнения HTTP-запросов к API для получения текущей цены BTC.
     ```bash
     pip install requests
     ```
   - `smtplib` (встроенная в Python библиотека): Для отправки email-уведомлений.
   - `twilio` (если понадобится отправка SMS):
     ```bash
     pip install twilio
     ```

3. **API для получения цены BTC**:
   - **CoinGecko API**: Бесплатный публичный API для получения информации о ценах криптовалют.
     - Документация: https://www.coingecko.com/en/api
   - **Binance API**: Еще один популярный API для получения данных о криптовалюте.
     - Документация: https://binance-docs.github.io/apidocs/spot/en/

#### Функции и логика:

1. **Функция получения цены BTC**:
   - Использует `requests` для обращения к API и получения текущей цены BTC.
   - Обрабатывает возможные ошибки и исключения, такие как временные сбои соединения или недоступность API.

2. **Логика мониторинга**:
   - Функция сравнивает текущую цену BTC с пороговым значением (80,000).
   - Осуществляет периодическую проверку через заданные интервалы (например, каждые 5 минут) с помощью цикла.

3. **Функция отправки уведомлений**:
   - В зависимости от выбранного метода, использует `smtplib` для отправки email или `twilio` для отправки SMS.
   - Проверяет успешность отправки уведомления и обрабатывает возможные ошибки.

#### Развертывание и поддержка:

1. **Автоматизация запуска**:
   - Использовать `cron` (на Linux) или планировщик задач (на Windows) для периодического запуска скрипта.

2. **Поддержка и обновления**:
   - Регулярно проверять и обновлять скрипт для совместимости с изменениями в API.
   - Вести мониторинг работы скрипта для своевременного выявления и исправления ошибок.

Эта спецификация позволяет разработать и развернуть скрипт, который будет надежно мониторить цену BTC и отправлять уведомления при падении цены ниже заданного порога.
