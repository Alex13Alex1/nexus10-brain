# Техническая спецификация для Todo List Manager с уровнями приоритета

## 1. Библиотеки Python с версиями

- **FastAPI (v0.95.2)**: Легковесный и быстрый фреймворк для создания RESTful API, который подходит для небольших проектов, таких как менеджер задач.
- **SQLAlchemy (v2.0.17)**: ORM для работы с реляционными базами данных, такими как PostgreSQL или MySQL, для хранения данных о задачах и их приоритетах.
- **Pydantic (v1.10.7)**: Для валидации данных и управления конфигурацией через типизированные модели.
- **Redis (v4.5.1)**: Для кэширования часто запрашиваемых задач, что уменьшает нагрузку на базу данных и ускоряет доступ.
- **uvicorn (v0.22.0)**: ASGI-сервер для запуска FastAPI-приложений.
- **dataclasses (встроенная библиотека Python)**: Для создания классов данных, которые могут использоваться для представления задач и их атрибутов.

## 2. API

- **RESTful API**: 
  - **Endpoint**: `/tasks`
  - **Методы**: 
    - `GET`: Получение списка задач с возможностью фильтрации и сортировки по приоритету.
    - `POST`: Создание новой задачи с указанием приоритета.
    - `PUT`: Обновление существующей задачи или её приоритета.
    - `DELETE`: Удаление задачи.
  - **Ответ**: JSON объект с данными о задачах.

## 3. Форматы данных

- **JSON**: Используется для передачи данных между клиентом и сервером через API. Это стандартный формат для RESTful API, обеспечивающий легкость интеграции и обработки данных.

## 4. Обоснование выбора

- **FastAPI**: Предоставляет высокую производительность и простоту использования, что делает его идеальным выбором для создания RESTful API без излишней сложности.
- **SQLAlchemy**: Позволяет легко взаимодействовать с базой данных, обеспечивая гибкость и мощные возможности ORM.
- **Pydantic**: Обеспечивает строгую валидацию данных и управление конфигурацией, что критично для обеспечения корректности входных данных.
- **Redis**: Используется для кэширования, что значительно ускоряет доступ к часто используемым данным и уменьшает нагрузку на основную базу данных.
- **dataclasses**: Встроенная библиотека Python, которая позволяет легко создавать классы данных для представления задач и их атрибутов.

Эти технологии и библиотеки были выбраны с учетом их стабильности, производительности и способности удовлетворять требованиям системы, обеспечивая при этом безопасность и оптимизацию ресурсов.
